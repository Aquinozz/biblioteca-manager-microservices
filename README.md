Este projeto é uma evolução do Biblioteca Manager monolítico.

Repositório original:
https://github.com/Aquinozz/biblioteca-manager-api

Objetivo:
- Migrar para arquitetura de microserviços;
- Utilizar Kafka para comunicação assíncrona;
- Utilizar OpenFeign/OpenAPI para comunicação síncrona;
- Implementar observabilidade e resiliência.
